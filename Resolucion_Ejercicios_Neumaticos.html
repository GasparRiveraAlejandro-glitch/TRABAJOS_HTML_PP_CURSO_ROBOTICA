🔷 EP-01: Garra Paralela – Selección
Masam = 0.5 kg (vidrio)
Aceleracióna = 3 m/s²
PresiónP = 5.5 bar = 550,000 Pa
Factor Seg.FS = 2.0
Rendimientoη = 0.82

(a) Fuerza Mínima Requerida

Freq = FS · m · (g + a) = 2.0 × 0.5 × (9.81 + 3) = 12.81 N
✓ Freq = 12.81 N

(b) Área Mínima del Pistón

Amin = Freq / (P · η) = 12.81 / (550,000 × 0.82) = 2.84 × 10⁻⁵ m² = 28.4 mm²
✓ Amin = 28.4 mm²

(c) Diámetro Mínimo

Dmin = √(4 · Amin / π) = √(4 × 2.84×10⁻⁵ / 3.1416) = 0.00601 m = 6.01 mm
✓ Dmin = 6.01 mm
📌 Selección comercial: cilindro Ø8 mm, Ø10 mm o Ø12 mm
🔷 EP-02: Garra Angular – Verificación
DiámetroD = 25 mm
PresiónP = 4 bar = 400,000 Pa
Rendimientoη = 0.78
Coef. Trans.K = 0.65
Masam = 2 kg
Aceleracióna = 5 m/s²

Paso 1: Fuerza Teórica

Fteo = P · (π/4) · D² = 400,000 × 0.7854 × 0.000625 = 196.35 N
✓ Fteo = 196.35 N

Paso 2: Fuerza Real en Mordaza

Fsuj = Fteo × η × K = 196.35 × 0.78 × 0.65 = 99.55 N
✓ Fsuj = 99.55 N

Paso 3: Fuerza Requerida

Freq = FS · m · (g + a) = 2.0 × 2 × (9.81 + 5) = 59.24 N
✓ Freq = 59.24 N

Paso 4: Verificación

Fsuj = 99.55 N ≥ Freq = 59.24 N
✅ SÍ ES APTA — La garra cumple con el requerimiento
FSreal = 99.55 / 59.24 = 1.68 | Pmin = 2.38 bar

🔷 EP-04: Mesa XY – Tiempo de Ciclo
QX20 L/min = 3.333×10⁻⁴ m³/s
QY12 L/min = 2.0×10⁻⁴ m³/s
DX50 mm
DY40 mm
Carrera X400 mm
Carrera Y200 mm

(a) Velocidades

vX = QX/AX = 3.333×10⁻⁴ / 1.9635×10⁻³ = 0.170 m/s
vY = QY/AY = 2.0×10⁻⁴ / 1.2566×10⁻³ = 0.159 m/s
✓ vX = 0.170 m/s | vY = 0.159 m/s

(b) Tiempos de Desplazamiento

tX = 0.4/0.170 + 0.02 = 2.373 s | tY = 0.2/0.159 + 0.02 = 1.278 s
✓ tX = 2.373 s | tY = 1.278 s

(c) Tiempo Total de Ciclo

Tciclo = tX + tY + tY = 2.373 + 1.278 + 1.278 = 4.929 s
✓ Tciclo = 4.93 s (ciclo parcial: X+ → Y+ → Y-)
Ttotal (incluyendo X-) = 2.373 + 1.278 + 1.278 + 2.373 = 7.302 s
📌 Ciclo completo (ida y vuelta): Ttotal = 7.30 s
